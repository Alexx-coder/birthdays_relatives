Это Бот напоминайка где будут храниться день рождения ваших родственников.
Текущие возможности:
-Количество родственников 8
-Выбор родственника
-Нахождение родственника с его именем и датой рождения
-Программа будет писать через сколько дней у данного родственника день рождение
-Программа будет писать в конце какая сегодня дата и сколько времени идёт данный день

Что в будущем может появиться:
-Регистрация/Вход
-Хранение имён и даты рождения ваших родственников
-Большее количество родственников
